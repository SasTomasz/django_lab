# Class-based views
I decided to use class-based way so code will be more elegant and more easy to maintain. 

Basic information about basic class base views can be found in 
[base views documentation](https://docs.djangoproject.com/en/6.0/ref/class-based-views/base/#base-views).

As I understand for now (2026-03-16) Django has a class-based views and class-based generic views. The latter are fresh
approach to solve some problems with class-based views.
